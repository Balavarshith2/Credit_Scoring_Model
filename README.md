Credit Risk Prediction System
Machine Learning | Classification | Finance Domain

This project is about predicting the risk involved in giving a loan to a customer. Instead of just focusing on accuracy, the idea here is to understand how data can help in making safer financial decisions.

1. Why this project?
When someone applies for a loan, banks need to decide : Will this person repay the loan or not?
Making the wrong decision can lead to financial loss.  
This project tries to solve that problem using machine learning.

2. What the model works on
The model looks at different details about a customer, such as:
- Income level  
- Loan amount requested  
- Credit history  
- Employment status  
- Other personal and financial factors  

Using this information, it predicts whether the applicant is **low risk or high risk.

3. How I approached the problem
Instead of directly training a model, I followed a step-by-step process:
- Started with exploring the dataset to understand patterns  
- Cleaned the data and handled missing values  
- Converted text-based data into numerical format  
- Selected important features affecting loan approval  
- Trained classification models  
- Compared results to see which model performs better

4. Running the project
- Clone this repository  
- Install dependencies using `pip install -r requirements.txt`  
- Open the notebook  
- Run the cells to see the full workflow

5. Takeaway
This project helped me understand how machine learning is used beyond theory especially in areas like finance where decisions have real impact.
